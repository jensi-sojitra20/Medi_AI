# 🧠 Medi AI

### AI-Driven Treatment Recommendation System

🚀 A smart healthcare web application that provides AI-based treatment suggestions and helps doctors generate prescriptions efficiently.

---

## 📌 Project Overview

Medi AI is a role-based web platform where:

* 👤 Patients can request AI-generated treatment recommendations
* 👨‍⚕️ Doctors can review and approve prescriptions
* 🛠️ Admin manages and verifies the system

The goal is to **improve healthcare efficiency using AI + automation**.

---

## ✨ Features

### 👤 Patient

* Register & Login
* Request AI treatment recommendations
* View prescriptions

### 👨‍⚕️ Doctor

* View patient requests
* Approve AI-generated prescriptions
* Give feedback

### 🛠️ Admin

* Manage users
* Verify doctors
* Monitor system

## ⚙️ Installation & Setup

### 1️⃣ Clone Repository

```
git clone https://github.com/jensi-sojitra20/Medi_AI.git
cd Medi_AI
```

### 2️⃣ Backend Setup

```
cd backend
pip install -r requirements.txt
python main.py
```

### 3️⃣ Frontend Setup

```
cd front end
npm install
npm run dev
```
